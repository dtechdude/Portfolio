What is a List?

In this article:

A List is a group of customers. Unlike Segments that are automatically managed by Engage based on customer data, you can add or remove customers from a List as you want.

Lists are useful for creating user or customer groups for newsletters, waitlist updates, and related campaigns.

How to add customers to a List

  1. You can import a CSV file containing your customer data via the Engage dashboard.

  2. You can copy and paste emails via the Engage dashboard.

  3. You can use the List API to add customers to (and remove customers from) a List.

  4. Each List has a subscription form you can copy, customize and embed on your HTML website.

What can you do with a List?

  1. Send email, SMS, push notifications to the List

  2. Trigger an automation when someone subscribes to the List

  3. Trigger an automation when someone unsubscribes from the List
